Instructions:

  • Choose your .exe file you want to change its looks in the start menu.

  • [Optional] Choose your windows tile image. If you don't choose an image, a windows tile will have default app icon. Supported formats: .png, .jpg

  • If you are not happy with the default background color of a tile, select your own by clicking on the color box.

  • Size checkboxes determine what tile sizes should be created. Functional only if you selected your own icon image.

Now you can click on the create button to make your requirements real. :)



What the program does:

In this section you can learn more about creation steps.
Exe file By this file, the application will know where the components should be saved.
Image file This is a working file and it determines the looks of a tile.
Size
checkboxes		 Determines what kind of tiles will be created. This is necessary only if you select an image path!
Color box By clicking on this box you can choose your own background tile color. Transparency, unfortunately, Windows does not support :(
Restore color A small icon near the color box will restore your selected color to default color.
Preview Near the Create button you can see a preview of a tile you want to create. This is visible only if you want to set your own icon.
Create The main button to start making your requirements real.
Console Every message from this application will be shown here.


About:

This program creates custom Windows 10 tiles. This copyright program has the Apache License 2.0 and was created by BenSvK. The application was intended to open-source usage.

Program features:

  • Creates the 'VisualManifestElements.xml' file inside of an exe file directory with user requirements. Such as background tile color, custom user image (70x70 or 150x150) or enables the title of an application on a large tile.

  • Creates the 'Icons' folder inside of the exe file directory with occasional images of the application. This folder is being created only if a user requires an own application image.

  • The program saves its local settings into the %AppData%\WindowsTileCreator folder as save.properties. This feature is smart for quick loading your previous settings.
